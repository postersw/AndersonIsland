4/26/17. FERRY TICKETING SYSTEM:
The TICKETS link on the main screen (row 2 column 3) now starts the new ferry ticketing app if you have it installed. If you don't have it installed, it takes to you a screen to install it. If you are using a PC or tablet, it takes you to the Ticketing Website.
4/26/17. CHURCHES:
A list of Anderson Island churches, and their service times, has been added under ISLAND INFORMATION LINKS -> CHURCHES.
4/26/17. CALENDAR:
The calendar on the Anderson Island Assistant lists ALL public events on the island. If an event is missing, please let us know through the 'Feedback' link.
